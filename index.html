구글 시트 연동 방법

  1. 아래 링크에서 템플릿 시트를 복사하세요 → 템플릿 열기 (예시 주소)
  2. 시트 메뉴에서 파일 → 공유 → 웹에 게시 클릭
  3. 형식을 CSV로 선택하고 게시 클릭
  4. 생성된 URL을 위 입력창에 붙여넣고 불러오기 클릭
시트 URL을 입력하거나 샘플 데이터로 먼저 확인해보세요.
